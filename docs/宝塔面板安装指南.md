# 宝塔面板一键安装 IDC 分销平台

> 总共 4 步：装软件 → 跑命令 → 配网站 → 守护进程

---

## 第一步：宝塔安装必要软件

宝塔面板 → 软件商店 → 搜索安装：

1. **PostgreSQL 16+**（记下密码）
2. **Redis 7+**
3. **Nginx**（一般已预装）
4. **Node.js 版本管理器** → 装 Node.js 22+

---

## 第二步：终端执行一键安装

宝塔面板 → 终端，粘贴执行：

```bash
cd /www/wwwroot && \
git clone https://github.com/Moon-Spirit/idc-platform.git && \
cd idc-platform && \

# 装编译工具
apt-get update && apt-get install -y cmake g++ libpq-dev libyaml-cpp-dev && \

# 配数据库
sudo -u postgres psql -c "CREATE USER idcapp WITH PASSWORD 'idcapp123';" 2>/dev/null
sudo -u postgres psql -c "CREATE DATABASE idc_platform OWNER idcapp;" 2>/dev/null
for f in backend/migrations/0*.sql; do
  sudo -u postgres psql -d idc_platform -f "$f" 2>/dev/null
done
sudo -u postgres psql -d idc_platform -c "GRANT SELECT, INSERT, UPDATE, DELETE ON ALL TABLES IN SCHEMA public TO idcapp; GRANT USAGE, SELECT ON ALL SEQUENCES IN SCHEMA public TO idcapp;" && \

# 编译后端（首次约3-5分钟自动下载依赖）
cd backend && mkdir -p build && cd build
cmake .. -DBUILD_POSTGRESQL=ON -DBUILD_REDIS=ON && make -j$(nproc) && \

# 构建前端
cd /www/wwwroot/idc-platform/frontend && npm install && npm run build
```

等待编译完成（约 3-5 分钟），看到 `✓ built in` 即成功。

---

## 第三步：配 Nginx 网站

宝塔面板 → 网站 → 添加站点：

| 设置 | 值 |
|------|-----|
| 域名 | 你的域名或 IP |
| 根目录 | `/www/wwwroot/idc-platform/frontend/dist` |
| PHP | 纯静态 |

添加后 → 设置 → 反向代理 → 添加：

```
代理名称: api
目标 URL: http://127.0.0.1:8080
```

---

## 第四步：守护后端进程

宝塔 → 软件商店 → 安装 **Supervisor管理器**

添加守护进程：

| 设置 | 值 |
|------|-----|
| 名称 | idc-platform |
| 运行目录 | `/www/wwwroot/idc-platform/backend` |
| 启动命令 | `export LD_LIBRARY_PATH=./deps/lib && IDC_DB_PASSWORD=idcapp123 ./build/idc-platform` |

---

## 完成 ✅

浏览器访问你的域名 → 自动进入安装向导 → 完成配置即可使用。

总耗时约 **10分钟**（编译占 5 分钟，实际动手 5 分钟）。
